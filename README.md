# Prepzo — AI Career Operating System for Universities

> Preparing students for the future of work using Artificial Intelligence.

## 🚀 About

Prepzo is an AI-powered placement ecosystem designed for universities. We replace guesswork with data-driven placement preparation — connecting students, universities, and recruiters through a single intelligent platform.

### The Problem
- 5 million graduates enter India's workforce every year
- Only ~45% are considered employable
- 70–80% fail in early interview rounds
- Universities have zero visibility into student readiness

### The Solution
- 🤖 **AI Interview Simulation** — hyper-realistic mock interviews tailored to each student
- 📊 **Placement Analytics** — department-wide dashboards for universities
- 🎯 **Skill Gap Analysis** — precision identification of weak areas + improvement roadmap
- 🔄 **Continuous Readiness Loop** — preparation that adapts all semester long

## 🛠 Tech Stack

- **React 18** + **Vite**
- **Three.js** + **@react-three/fiber** + **@react-three/drei** — 3D hero scene
- **Framer Motion** — animations
- **Tailwind CSS** — styling
- **Supabase** — contact form backend
- **Lucide React** — icons

## 📦 Getting Started

```bash
npm install
npm run dev
```

Create a `.env` file:
```
VITE_SUPABASE_URL=your_supabase_url
VITE_SUPABASE_ANON_KEY=your_anon_key
```
